# Epilobe_Standalone

Pour faire fonctionner Epilobe_Standalone il faut modifier paths.json et Epilobe\params.json

- params.json contient les données générées quand on fait une carte des vents depuis Santoline
- je vous laisse lire mon paths.json et le changer on fonction de votre configuration (asctojsonparser_path c'est le chemin vers l'executable de AscToJsonParser et wind_map_generation_path est l'endroit où vous voulez que la carte des vents (truc.json) se génère).
